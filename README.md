<h1 align="center"><b>akaOS</b></h1>

<p align="center">
  akaOS is a <b>free and open-source operating system</b> based on Arch Linux.
</p>

## Download
You can download the latest ISO from the [akaOS Releases](https://github.com/akaoperatingsystem/akaOS/releases) <br>
**NOTE:** Installing akaOS 1.0.0 (Nova) requires an active internet connection.
